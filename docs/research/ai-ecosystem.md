# AI 音乐制作生态全景

> 调研日期：2026-06-18
> 调研方式：mimo search 联网搜索 + Web Search + 页面抓取

## 结论先行

AI 音乐制作目前形成了**三个明确的层次**，我们处在第三层的起步阶段：

```
层次 1: AI 生成素材 → 人挑选使用（Suno/ORB/Pack Generator）
层次 2: AI 当问答助手 → 人自己操作（Gopher/ChatGPT）  
层次 3: AI 直接操控 DAW → 人审核微调（MCP 方案）← 我们在这里
```

**核心判断：** 层次 1 和 2 已经成熟且竞争激烈，层次 3 刚刚起步，仅有几个社区项目在探索。这是我们的机会窗口，也是我们的挑战——没有成熟路径可抄。

---

## 一、FL Studio 官方 AI 布局

Image-Line 从 2025 版开始全面拥抱 AI，但定位是"辅助"而非"操控"：

### Loop Starter — 素材生成器
- 按风格（电子/古典/boom bap/雷鬼/phonk/afrobeats）生成循环音轨堆栈
- 自动生成可编辑的 Step Sequencer 音轨
- 支持一键排列到播放列表
- **本质：** 层次 1 的产品，帮你找灵感起点

### Gopher — AI 问答助手
- 基于 FL Studio 官方手册/FAQ/知识库训练
- 支持多语言聊天式问答
- **本质：** 层次 2 的产品，替代翻手册查文档

### AI 分轨 — 音频分离
- 21.2.2 新增，导入采样可智能分轨
- 可路由到单独 Mixer 轨道（2025.2 增强）
- **本质：** 单向的音频分析，不能反向写入

### 官方判断
Image-Line 的策略很清楚：**让 FL Studio 更容易上手，但不改变"人主导创作"的范式**。他们不会做"AI 替你编曲"，因为那会动摇 DAW 的产品定位。

## 二、第三方 AI 工具生态

### 素材生成类（层次 1）

| 工具 | 原理 | 与 FL Studio 的关系 | 评价 |
|------|------|-------------------|------|
| **ORB Producer Suite** | AI 生成和弦/旋律/Bass/琶音 MIDI | VST 插件在 FL 内运行，¥199 | 最接近"AI 编曲"的成熟产品 |
| **Pack Generator** (Output) | 文字提示 → 4 个采样包（25 个 loop） | 下载后导入 FL | 素材质量不错，但不够精准 |
| **SoundRaw** | 分段落调节乐器/节奏/音高 | 导出音频后导入 FL | 日本团队，pro 模式可细分 5 轨 |
| **Riffusion** | 文生图→图转音乐（stable diffusion） | 导出音频后导入 FL | 免费开源，效果一般 |
| **Suno** | 端到端一句话生成完整歌曲 | 导出后可导入 FL 二次编辑 | 最成熟但最不可控 |

### 问答辅助类（层次 2）

| 工具 | 说明 |
|------|------|
| **Gopher** | FL Studio 内置，只回答 FL 操作相关问题 |
| **ChatGPT/Claude** | 通用 AI，可以给编曲建议但不能操控 DAW |

### Agent 操控类（层次 3）

| 项目 | 状态 |
|------|------|
| **flai-mcp** | Alpha，功能最全，SysEx 双向通信 |
| **fl-studio-mcp** | 较新，Piano Roll 精度高 |
| **flstudio-mcp-plus** | 🚧 规划中（我们） |

## 三、市场空白与机会

### 没人做的事

1. **统一的全功能 Agent** — 现有方案要么只管 Mixer（flai-mcp 的旋律写入弱），要么只管 Piano Roll（fl-studio-mcp 的 Mixer 控制缺失）
2. **中文优先** — 所有项目都是英文，中文音乐术语理解是空白
3. **"写完直接导出"** — 从 AI 操控到分轨导出的完整闭环
4. **风格模板** — "progressive house 四小节"这种结构化 prompt 没有标准化

### 已经成熟（不需要我们做的）

1. AI 生成素材（Suno/ORB 已经很卷）
2. AI 问答助手（Gopher 已内置）
3. 音频分轨（FL 官方已支持）

---

## 四、对 flstudio-mcp-plus 的定位

我们不是在跟 Suno 竞争，也不是在做第二个 Gopher。

**我们要做的：**
> 让 AI 成为 FL Studio 里的"虚拟制作人"——制作者用自然语言描述意图，AI 直接在 DAW 里执行操作，制作者审核微调。

**核心差异化：**
- 全功能统一（Mixer + Piano Roll + Pattern + 插件 + 导出）
- 中文原生
- 完整闭环（写入 → 混音 → 导出）

**一句话：** 别人给你素材让你挑，我们让 AI 帮你炒菜。
