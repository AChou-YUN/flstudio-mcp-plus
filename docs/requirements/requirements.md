# 📋 需求文档

## 功能需求

### P0 — 核心功能（必须）

| 模块 | 功能 | 说明 |
|------|------|------|
| Transport | 播放/暂停/停止 | 基本播放控制 |
| Transport | 录音控制 | 开始/停止录音 |
| Transport | 位置/速度 | 设置播放位置、速度 |
| Mixer | 音量/声像 | 每个轨道独立控制 |
| Mixer | 静音/独奏 | 轨道状态控制 |
| Channel | 通道列表 | 获取所有通道信息 |
| Channel | 通道属性 | 音量、声像、名称、颜色 |
| Piano Roll | 音符操作 | 添加/删除/清空音符 |
| Piano Roll | 和弦操作 | 一键添加和弦 |
| Piano Roll | 状态读取 | 获取当前音符状态 |

### P1 — 重要功能

| 模块 | 功能 | 说明 |
|------|------|------|
| Plugin | 参数控制 | 读取/设置插件参数 |
| Plugin | 预设切换 | 上一个/下一个预设 |
| Channel | 步进音序器 | 读取/设置 grid bits |
| Channel | 路由 | 通道路由到 Mixer 轨道 |
| Mixer | 录音臂 | 轨道录音准备 |
| Mixer | 立体声分离 | 调整立体声宽度 |

### P2 — 创新功能（差异化）

| 模块 | 功能 | 说明 | 难度 |
|------|------|------|------|
| Plugin | 加载插件 | 突破 API 限制 | 🔴 高 |
| Pattern | 创建 Pattern | 突破 API 限制 | 🔴 高 |
| 中文 | 中文自然语言 | 国内用户友好 | 🟡 中 |
| AI | 智能编曲建议 | 基于上下文推荐 | 🟡 中 |

## 非功能需求

| 维度 | 要求 |
|------|------|
| 延迟 | 操作响应 < 2s |
| 兼容性 | FL Studio 20.7+ |
| 平台 | macOS + Windows |
| 稳定性 | 7×24 不崩溃 |
